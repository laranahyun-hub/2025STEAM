🌿 아로마 롤온 오일 만들기 체험🌿

나만의 향으로 힐링하는 시간

1️⃣ 준비물

  • 에센셜 오일: 라벤더, 페퍼민트, 스윗오렌지, 티트리, 레몬그라스, 유칼립투스
  • 베이스 오일: 해바라기씨유 10ml
  • 10ml 롤온 용기
  • 라벨지 및 펜

2️⃣ 제작 방법

  1. 에센셜 오일 10방울을 용기에 넣습니다.
    (피부용은 1% 이하 희석 권장 — 10ml 기준 약 2방울이 1%)
  2. 베이스 오일(해바라기씨유) 10ml를 추가합니다.
  3. 롤 부분을 끼우고 가볍게 흔들어 잘 섞어줍니다.
  4. 제작 날짜와 향 블렌딩 정보를 라벨에 기록하여 보관합니다.

3️⃣ 향 분류 및 추천 블렌딩

오일 계열 향 특징 / 효능 추천 블렌딩
라벤더 플로럴 부드럽고 안정감 있는 향, 숙면·불안 완화 스윗오렌지 (숙면) / 티트리 (마음 안정)
스윗오렌지 시트러스 달콤하고 따뜻한 향, 기분 전환, 스트레스 완화 라벤더 (숙면) / 레몬그라스 (피로 회복)
페퍼민트 허벌 / 멘톨 시원한 향, 집중력 향상, 두통 완화 유칼립투스 (호흡기 개선) / 라벤더 (스트레스 완화)
티트리 허벌 / 우디 항균·항바이러스, 피부 트러블 완화 레몬그라스 (공기 정화) / 라벤더 (진정)
레몬그라스 시트러스 / 허벌 살균, 방취, 염증 완화, 공기 정화 티트리 (살균 강화) / 스윗오렌지 (밝은 향)
유칼립투스 우디 / 멘톨 항바이러스, 호흡기 완화, 상쾌함 페퍼민트 (두통·집중) / 스윗오렌지 (균형)

4️⃣ 학생용 추천 블렌딩 예시

상황 / 목적 블렌딩 조합 비율(10방울 기준) 설명
숙면 / 긴장 완화 라벤더 + 스윗오렌지 2 : 1 라벤더 진정 효과 + 오렌지 따뜻한 향 → 심신 안정
공부 / 집중력 향상 페퍼민트 + 유칼립투스 + 레몬그라스 1 : 1 : 1 멘톨 계열 향으로 머리 맑게 → 집중력 향상
스트레스 완화 / 휴식 라벤더 + 스윗오렌지 + 티트리 2 : 1 : 1 마음 안정 + 상쾌함 → 휴식에 도움
불안 / 예민함 진정 라벤더 + 티트리 2 : 1 정화 작용 + 진정 효과 → 마음 안정
피로 회복 / 활력 부여 스윗오렌지 + 레몬그라스 + 페퍼민트 2 : 1 : 1 시트러스 향 활력 + 페퍼민트 청량감 → 피로 회복
두통 / 코막힘 완화 유칼립투스 + 페퍼민트 1 : 1 멘톨 계열 향으로 머리 맑게 + 호흡기 개선
공기 정화 / 정신 집중 티트리 + 유칼립투스 + 라벤더 1 : 1 : 1 항균 + 안정 효과 → 쾌적한 학습 환경
